HTML

  • To give the content structure
  • Made of elements which work together to provide content and the basic framework for all pages on the web
  • Attributes are always defined in the beginning element, never the closing element
  • An element is the entire element from opening tag to closing tag
  • The head element contains information about the webpage.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements
  • The element that holds all the children are known as the parent element
  • A tag only appears inside the angle brackets

CSS

  • To give the content style
    • li>Inline CSS is when a developer wants to style an element directly in the HTML file
  • An internal syle sheet contains CSS rules for the webpage in the head section of the HTML file
  • An external style sheet allows developers to keep all their CSS rules in a seperate file
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • A CSS box model consists of margins outsid the element, and the element's borders, padding, and content

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m: commits your staged content as a new commit snapshot
  • git pull: fetch and erge any commits from the trackinf remote branch
  • git push: transmit local branch commits to the remote repositiory branch
  • git add: add a file as it looks now to your next commit

JavaScript

  • To handle logic and allow users to interact with a website
  • A variable is a named container that allows us to store data in our code.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • Control flow is the order in which a computer executes code in a script.
  • Code runs from the first statement at the top of the page, down to the last statement at the bottom of the page
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
